Escaneo y reconocimiento

  • Nmap - Escáner de red para descubrir hosts y servicios.
  • Wireshark - Analizador de tráfico de red.
  • Recon-ng - Framework para reconocimiento web.

Explotación y pruebas

  • Metasploit - Plataforma para desarrollar y ejecutar exploits.
  • Burp Suite - Proxy para pruebas de seguridad web.
  • SQLmap - Automatiza la detección y explotación de inyecciones SQL.

Cracking y descifrado

  • John the Ripper - Herramienta de cracking de contraseñas.
  • Hashcat - Crackeo de contraseñas usando GPU.

Análisis y post-explotación

  • Maltego - Herramienta para análisis forense y de inteligencia.
  • Autopsy - Plataforma forense digital para análisis de discos y archivos.

Automatización y scripting

  • Python - Lenguaje para automatizar tareas y crear scripts.
  • Bash - Shell scripting para Linux y sistemas Unix.
